# Data Visualization Trainee Early Remote Internship – DePaul Outreach Analysis

## Project Overview
Analyzed 7,543-record dataset for DePaul graduate admissions outreach. Key tasks: Data dictionary, cleaning (missing/duplicates), EDA insights, interactive Power BI dashboard, recommendations.

## Key Achievements
- Identified trends: India 61% volume, 68% response rate.
- Built multi-page dashboard with KPIs (response 57.7%), breakdowns, trends.
- As Project Scribe, documented team processes.
